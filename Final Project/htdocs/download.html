Download Bootstrap

To begin using Bootstrap, you must download the current version of the framework. Go to Get Bootstrap and select the "Download Bootstrap" button near the top of the page. This will download the source code for a set of Bootstrap style sheets to your computer, allowing you to copy them into a 'css' or related folder of any website you’re developing.

Once the style sheets have been allocated to your project, make sure to reference them in the head section of each page. The framework is then ready to be called in your HTML.

To understand how to apply the styles available in each of the style sheets, refer to the Getting Started page on the Bootstrap website. Additionally, the site explains how to use the CSS settings to customize your site as you see fit.
