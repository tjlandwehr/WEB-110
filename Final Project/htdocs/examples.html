Examples of Bootstrap

Grid System

One of the basic elements for layout design is the grid system. Typically, the viewport is divided by using the div tag along with CSS to layout elements where desired. Bootstrap contains its own grid layout system, consisting of a 12‐column grid with two versions: default and fluid. The default grid system uses a 12‐column grid that renders out at a width of 940 pixels, while the fluid grid system uses percentages for column widths rather than fixed pixels.

Responsive Design

Creating a Web page which adapts its layout automatically to the screen size and resolution of devices such as smartphones, PCs, and tablets is considered a requirement today. Bootstrap offers a flexible screen layout with its grid system and responsive design. The grid size changes accordingly with the size of the screen as shown in the table below.

Changed Grid Size by Device
Label Layout width Column width Gutter width
Smartphones 480 pixels and lower Fluid columns, no fixed widths
Portrait tablet PCs 767 pixels and lower Fluid columns, no fixed widths
Landscape tablet PCs 768 pixels and higher 42 pixels 20 pixels
Default 980 pixels and higher 60 pixels 20 pixels
Wide‐screen monitor 1200 pixels and higher 70 pixels 30 pixels
