Responsive Images

For images to be responsive (so they will rescale to suit the width of the parent) use the img-responsive class.

Unordered List

Unordered lists are used to markup lists where list items do not require numbering.

  • Curabitur ornare rutrum libero, at dapibus tellus efficitur ac.
  • Aenean eleifend finibus sem nec mollis.
  • Quisque volutpat cursus sollicitudin.
  • Aenean mattis neque vel augue molestie, semper sollicitudin lectus tempus.
  • Nullam quis nisi viverra enim placerat blandit.

Ordered List

Ordered lists are used to markup lists where list items require numbering.

  1. Etiam euismod pharetra quam, ac mattis mi pretium at.
  2. Maecenas urna dui, fringilla a accumsan vitae, ullamcorper id nisi.
  3. Vestibulum ante ipsum primis in faucibus orci luctus et ultrices posuere cubilia Curae; Duis in lacus ut est aliquam eleifend.
  4. Pellentesque placerat pretium orci, nec accumsan magna rhoncus sit amet.
  5. Quisque et rhoncus dolor.

Unstyled List

You can make the list unstyled using the list-unstyled class.

  • Interdum et malesuada fames ac ante ipsum primis in faucibus.
  • Nulla feugiat tempor massa vitae euismod.
  • Aliquam dapibus ultrices efficitur.
  • Donec egestas varius purus, ut ultrices metus ullamcorper quis.
  • Aenean congue ac dui nec pellentesque.

Inline List

You can make the list display inline using the list-inline class.

  • One
  • Two
  • Three
  • Four

Blockquotes

The blockquotes element is used to markup long quotations. Ideally, content inside the blockquote should be wrapped in paragraph elements.

The real names of our people were destroyed in slavery. The last name of my forefathers was taken from them when they were brought to America and made slaves, and then the name of the slave master was given, which we refuse, we reject that name today and refuse it. I never acknowledge it whatsoever.

Malcolm X
